# KronosCMS WordPress-Parity Roadmap

Goal: bring KronosCMS close to a familiar WordPress admin experience while keeping the cleaner Kronos architecture.

## Completed in this pass

- Split Posts and Pages into separate dashboard areas.
- Add a WordPress-style menu editor for header/footer navigation.
- Add DB-backed menu storage and public menu rendering.
- Add taxonomy management for categories and tags.
- Connect post categories and tags to the content editor.
- Add a classic content textarea so posts/pages can have body content outside the builder.
- Add a Media Library foundation with upload support and copyable asset URLs.
- Expand the dashboard sidebar to include Media, Categories, Tags, and Menus.
- Add DB-backed media attachment records with alt text and captions.
- Add featured image fields to posts/pages and render featured images publicly.
- Add public `/post/{slug}` routing so post menu links resolve correctly.
- Add menu move up/down controls that update item order fields.
- Add full drag-and-drop ordering to Menus.
- Add nested menu items with public dropdown rendering.
- Add editor statuses for draft, published, scheduled, private, and archived content.
- Add preview support for unpublished content.
- Add revision snapshots for edited posts/pages.
- Add local browser autosave and quick classic editor insert buttons.
- Add media usage tracking and attachment detail pages.
- Add bulk status actions and screen-size controls to content list tables.
- Add permalink settings foundation.
- Add theme/customizer settings for logo, typography, and header/footer layout.
- Add plugin compatibility checks in Marketplace cards.
- Add WordPress-style role/capability aliases for administrator, editor, author, contributor, and subscriber.
- Add import/export foundation for settings.

## Next high-impact steps

- Add true Quick Edit rows for title, slug, status, and publish date.
- Add restore/revert actions for stored revisions.
- Extend import/export to posts, pages, menus, media metadata, and taxonomies.
- Add comments/moderation if public blogging becomes a priority.
- Add a plugin update screen with installed plugin inventory and rollback notes.
- Add richer customizer live preview for logo, colors, typography, and header/footer layout.
- Add scheduled publishing automation so due scheduled posts are promoted automatically.
